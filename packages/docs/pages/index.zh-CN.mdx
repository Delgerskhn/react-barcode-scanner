## React Barcode Scanner

一款基于 `Barcode Detection API` 和 `zbar.wasm` 的轻量化扫码器

### 特性

- 基于浏览器`Barcode Detection API`规范
- Polyfill 使用 `zbar wasm` 库，大小仅~230kb
- 基于React Hooks, 提供高自定义能力
- 使用 TypeScript 构建，提供完整的类型定义文件
